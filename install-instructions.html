About The Project

PortableMC:

Installation of Minecraft Java Edition, easy to setup and use:

Getting Started
To get the program up and running follow these simple steps.


Prerequisites
If Using The Java Release You Need:
Java
OR
Jportable
Normal Release Requires Windows Or a Windows (Mac and Linux Are Planned For The Future)

Installation
Java: Download And Run PortableMC-Java.jar
Normal: Download And Run PortableMC.exe
